detect_face Ver 1.0.0
=========
---------
顔を見つけて保存するだけ。  
中身はOpenCVだけ。  

使い方
---------
src/hoge/image.jpg の形になるように変換前の画像を入れる。  
dst/hoge/image_1.jpg の形で出力される。  

注意点
---------

* 一応顔認識部分から1/8だけ大きい領域を取っているが邪魔なら消せばいい。
